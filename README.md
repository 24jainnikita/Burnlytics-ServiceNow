# Burnlytics 🔥

A Smart Productivity & Burnout Detection System built on the ServiceNow platform.

---

## 📌 Project Overview

Burnlytics is a ServiceNow-based application designed to monitor employee workload, calculate burnout risk dynamically, and automate critical alerts for management.

The system uses Business Rules, Flow Designer automation, dashboards, and custom tables to provide real-time burnout analytics.

---

## 📊 Burnout Score Logic

Burnout score is calculated using:

Burnout Score = (Active Tasks × 5) + (High Priority Tasks × 10)

### Burnout Zones

- Safe Zone → score < 30
- Warning Zone → score between 30–59
- Critical Zone → score ≥ 60
